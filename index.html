THREEx three.js extensions vendor.js style

THREEx is a way to write three.js extension ala vendor.js. So as little dependancy as possible. It is modules to help you write three.js stuff, not a framework.

Existing Extensions

Here is a list of the existing extensions. Feel free to contribute your own. Just do a pull request , edit this file, copy an existing line and fit it to your own needs. Wanna try live in the browser ? There is a threex playground waiting for you! If you want to create your own, here is three.sample to start with.

threex.glowkeycolor

 It provides a glow based on key-color. There is a single scene which is rendered only once. It all happens in screenspace.

threex.skymap

 It is three.js extension to use the use of skymaps. Additionnaly it provides samples of cubemaps taken in three.js examples.

threex.keyboardstate

 It is a three.js extension to keep the current state of the keyboard. It may be queried at any time. This is convenient in loop driven case, like games.

threex.depthoffield

 It provides a tunable depth of field to your scene. It gives it a nice bokeh effect. It is very easy to include. It provides realistic physics in your 3d.

threex.cannonjs

 It provides realistic physics in your 3d. It is a wrapper to ease access between cannon.js and three.js.

threex.basiclighting

 It provides various ligthings e.g. three point lighting or a sunset lighting. This is for educational purpose or starting your project fast.

threex.windowresize

 It is a three.js extension to help you handle window resize.

threex.htmlmixer

 It is a three.js extension to seamlessly integrate dom elements in your webgl. It is from "Mixing HTML Pages Inside Your WebGL" on learningthree.js blog.

threex.noiseshadermaterial

 It is a three.js extension which provide a THREE.ShaderMaterial preconfigured to display 3d noise. It is made out of the webgl noise from ashima.

threex.nyancat

 It is a three.js extension to get a nyancat all over. It is made out of the nyancat anonymous webgl demo. It is released under MIT license.

threex.geometricglow

 It is a three.js extension to make any object glow at geometric level.

threex.planets

 It helps you to display all the planets from the solar system. It is on planetpixelemporium. As a cherry on the cake, it has a 'atmosphere material'.

threex.proceduralcity

 It is a three.js extension to display a fully procedural city based on "city" , a demo recently released by @mrdoob.

threex.rendererstats

 It is a three.js extension to display realtime informations about THREE.WebGLRenderer. It is widely inpired from @mrdoob stats.js.

threex.transparency

 It is a three.js extension to easily handle transparency. It is released under MIT license.
